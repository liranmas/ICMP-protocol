Message Sequence

Message sequence diagram example

In the first part of the sequence diagram, the source host sends data packets. When it arrives at the router, if there is an error with the packet, such as too large, an ICMP error message will be sent back to inform the source host that that packet was not successfully delivered.


In the second part of the sequence diagram, the source host sent data packets; when it arrives at the router, the router will check if there is an error. It will route to the destination host and perform another error checking. If there is an error in the packet, it will be sent back to the router to route the ICMP error message back to the source host.
